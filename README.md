# 家事代行サービス

## 概要
猫専門ペットシッターサービスのランディングページです。

## 使用技術
- HTML
- CSS
- JavaScript
- jQuery
- GitHub

## 工夫した点
- スマホ、タブレット、PC対応のレスポンシブデザイン
- ハンバーガーメニューの実装
- 料金表を横スクロール対応にして視認性を向上
- お問い合わせボタンを目立たせる導線設計

## 苦労した点

- CSSアニメーションの実装
